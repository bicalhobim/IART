INTRODUÇÃO AOS REQUISITOS TÉCNICOS

Objetivo: Este arquivo tem como objetivo explicar o que são requisitos técnicos e orientar sobre como o agente GPT deve utilizá-los no desenvolvimento de projetos em BIM.

O que é um Requisito Técnico:
Um requisito técnico (RT) é um documento que especifica as condições, critérios e parâmetros que um projeto, produto ou processo deve atender para estar em conformidade com as normas e regulamentos aplicáveis. Em engenharia, um RT é fundamental para garantir a qualidade, segurança e eficiência dos projetos, produtos ou processos.

Estrutura e Formatação:
- Objetividade e clareza são essenciais.
- Utilizar vocabulário técnico apropriado ao contexto das normas de engenharia.
- A escrita deve ser formal e direta, evitando ambiguidades.
- Cada seção deve ser claramente definida e separada por títulos descritivos.

Como Utilizar:
- Os arquivos de requisitos técnicos devem ser consultados conforme a necessidade do projeto em questão.
- O agente deve adaptar os requisitos técnicos de acordo com o contexto específico de cada projeto.
- Os exemplos fornecidos nos arquivos podem ser modificados para se adequar melhor às especificidades do projeto.

Estrutura de uma RT:
- Objetivo e Requisitos de Valor
- Usos do Modelo BIM
- Informações Mínimas
- Marcos Técnicos de Projeto em BIM
- Avanço do Modelo
