USOS DO MODELO BIM

Objetivo: Definir os Usos do Modelo BIM que compõem a finalidade do modelo desenvolvido pela contratada.

Lista de Usos Mínimos:
Item 1: Projeto Autoral
Descrição: Elaboração do projeto em BIM utilizando softwares autorais BIM para desenvolver um modelo 3D com informações atribuídas aos objetos paramétricos. Esse uso é de fundamental importância, pois é a base para os demais usos BIM descritos a seguir.
Prioridade: Alta
Etapa do Projeto: PB (Projeto Básico)

Item 2: Documentação 2D
Descrição: Extração de documentos 2D a partir de modelos BIM utilizando recursos de anotação inteligentes vinculados aos elementos, refletindo alterações feitas nos modelos.
Prioridade: Alta
Etapa do Projeto: PB (Projeto Básico)

Item 3: Detecção de Interferências
Descrição: Uso de modelos BIM (.NWD) para coordenar diferentes disciplinas e identificar/antecipar a resolução de possíveis interferências entre os elementos virtuais antes da fabricação ou construção.
Prioridade: Alta
Etapa do Projeto: PB (Projeto Básico)

Item 4: Reunião Design Review e Design Técnico
Descrição: Revisar um modelo de informações de construção com as partes interessadas do projeto para obter feedback e validar o design, a construção e aspectos técnicos do projeto.
Prioridade: Média
Etapa do Projeto: PB (Projeto Básico)

Item 5: Extração de Quantitativos
Descrição: Uso de modelos BIM para calcular a quantidade de elementos e materiais construtivos, alimentando as Planilhas de Quantidades conforme os Critérios de Medição de Serviço (CMS).
Prioridade: Alta
Etapa do Projeto: PB (Projeto Básico)

Item 6: Planejamento 4D
Descrição: Uso do modelo BIM vinculado ao cronograma de implantação do projeto para revisão dos processos construtivos durante a etapa de projeto.
Prioridade: Média
Etapa do Projeto: PB (Projeto Básico)

Exemplo de Aplicação:
Esses usos garantem que todas as fases do projeto, desde a concepção até a execução, sejam integradas e coordenadas eficientemente utilizando o modelo BIM.
